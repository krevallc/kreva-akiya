# 依頼：空き家「物件詳細ページ」のUIデザイン（高忠実度モックアップ）

## 背景
日本の不動産会社 KREVA LLC の空き家検索サイト（kreva.co.jp/akiya/）。
検索一覧ページは既に以下のデザイントークンでリデザイン済み。**今回はその詳細ページを、同じトークンで統一感のあるデザインにしたい。**

## 確定済みデザイントークン（必ずこれに従うこと）
- Color: accent `#b3541e`（テラコッタ）/ ink `#211d19` / sub `#6f6a63` / line `#e7e3dc` / bg `#faf9f6`
- バッジ: NEW `#1f8a5b` / 価格応談 `#5a5650` / 調整区域 `#d07f1f` / 災害想定 `#c2413a` / KREVA=accent
- Type: 'Noto Sans JP'。11/12/13/14px=メタ・ラベル、16px=小見出し、22px=価格・H2。**価格・見出しは weight 900**。本文行間1.9
- Radius: pill/ボタン 999px、カード 16px、ポップアップ・バナー 14–20px、バッジ 6px
- Shadow: hover `0 10px 28px rgba(28,24,18,.12)`、モーダル `0 12px 32px rgba(0,0,0,.22)`
- CTAバナー: accentの淡色背景（#f8ece3系）＋角丸20px
- Breakpoint: 820px

## ページに含める要素（現行機能・すべて維持）
1. **パンくず**（空き家検索 > 高梁市 > この物件）
2. **ヘッダー部**：タイトル（例「高梁市成羽町下原の空き家（No.720）」）・所在地・**価格（大きく・weight900）**・状態バッジ（NEW/調整区域/災害想定 等）
3. **フォトギャラリー（このページの主役・要件は下記）**：写真は最大8枚
4. **2カラム構成**（メイン+右サイドバー、モバイルは1カラム）
   - メイン：物件概要テーブル（価格/所在地/土地・建物面積/間取り/構造/築年/**耐震(推定)**）／用途・規制（区域区分※調整区域は警告表示・用途地域・建蔽率/容積率）／災害リスク（該当項目・液状化傾向）／生活・周辺（学区・最寄り医療機関/保育園/避難場所と距離・将来人口）／説明（ライフライン等の自由文）／**周辺地図**（Leaflet・ハザードレイヤ切替つき・カラム幅・高さ440px程度 → プレースホルダ画像でよい）
   - サイドバー：相場の目安（最寄り地価・固定資産税概算）／**相談CTA**（お問い合わせフォームへ）／出典・最終確認日（元ページリンク）
5. （任意・歓迎）「近くの物件」カード3件（検索ページのカードデザインを流用）

## ★フォトギャラリーの必須要件（最重要）
- 現状の問題：サムネクリックで**新しいタブに画像が開いてしまう** → これを廃止
- **クリックしたらページ遷移せず、その場でライトボックス（全画面オーバーレイ）が開く**こと：
  - 拡大画像を大きく表示、背景は暗転
  - **左右矢印・キーボード←→・スワイプ・スクロール（ホイール）で次/前の写真へ移動**
  - 「3 / 8」のようなカウンター表示
  - ✕ボタン・ESCキー・背景クリックで閉じる
  - 下部に横並びサムネイル（現在位置ハイライト）があると良い
- ギャラリー本体（ページ内）のレイアウトは提案に任せる（例：1枚目大＋サムネ帯、または2×2モザイク＋「すべての写真(8)」ボタン等、Airbnb/Zillow風）
- **ライトボックスはモックアップ内で実際に動くJSを実装すること**（この挙動仕様のまま実装に移植するため）

## 制約
- WordPress＋**素のJS/CSSで実装**（React等のフレームワーク不可。ライトボックスも依存ライブラリなしのvanilla JSで）
- 写真は外部サイト直リンクで**縦横比バラバラ** → object-fit: cover / ライトボックス内は contain
- 日本語UI・価格は「万円」表記
- 画像下に出典表記が必要（「画像出典：高梁市空き家バンク（元ページより）」）

## 成果物
1. 自己完結の単一HTMLファイル（CSS/JSインライン・外部依存なし・ダミー写真はプレースホルダ可・**ライトボックス動作込み**）
2. デスクトップ(1280px)とモバイル(390px)対応
3. ファイル冒頭コメントにデザイントークンと**ライトボックスの挙動仕様**を記載
